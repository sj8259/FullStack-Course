# Project 9 — React Core Concepts Demo (CDN)

This project demonstrates core React ideas using a no-build setup:
- Components and JSX
- Props
- State and Events
- Composition
- Controlled Forms

How to run:
1. Open `index.html` in your browser (double-click).
2. No local server or build step required; React is loaded via CDN.

Structure:
- `index.html` — mounts the app, defines components using `<script type="text/babel">`
- `styles.css` — basic styling
- `script.js` — placeholder for future non-React JS if needed
