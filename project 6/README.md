# To Do List Application

A simple and elegant To Do List application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Add Tasks**: Type a new task in the input field and click "ADD" or press Enter
- **Toggle Completion**: Click on the checkbox to mark tasks as complete/incomplete
- **Delete Tasks**: Click the ⊗ button to remove tasks with a smooth animation
- **Responsive Design**: Works on both desktop and mobile devices
- **Visual Feedback**: Smooth transitions and hover effects

## How to Use

1. **Open `index.html`** in your web browser
2. **Add a new task**: Type in the input field and click "ADD"
3. **Mark as complete**: Click the checkbox (☐) to mark a task as done (☑)
4. **Delete a task**: Click the ⊗ button to remove a task

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and visual design
- `script.js` - Interactive functionality
- `README.md` - This documentation

## Design Features

- **Orange Background**: Vibrant solid orange background as shown in the design
- **Cream Containers**: Elegant cream-colored task containers with subtle shadows
- **Green Elements**: Green checkboxes and add button for visual consistency
- **Modern UI**: Rounded corners, smooth transitions, and hover effects

## Browser Compatibility

Works in all modern browsers that support ES6 JavaScript and CSS3 features.

## Getting Started

Simply open the `index.html` file in any modern web browser. No server setup or dependencies required!
